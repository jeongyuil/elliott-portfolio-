1. 프로젝트 개요

앱 소개

Bamtory는 어린이를 위한 인터랙티브 영어 학습 앱입니다. 아이들은 캐릭터 "Luna"를 도와주며 자연스럽게 영어를 배웁니다.

핵심 기능:
  • 스토리 기반 미션 학습
  • 음성 인식 기반 스피킹 연습
  • 게이미피케이션 (레벨, 스타, 업적)
  • AI 튜터 "Popo"의 실시간 피드백

주요 캐릭터

🌙

Luna (루나)

영어를 배우고 싶은 귀여운 토끼.
아이들이 도와줘야 하는 주인공.

🦜

Popo (포포)

AI 튜터 앵무새.
힌트를 주고 격려하는 조력자.

5. 사용자 흐름 (User Flow)

미션 완료 플로우

🏠 홈 → 미션 선택 → 프리뷰 → Start → 🎤 음성 입력 → 피드백
성공: 피드백 → Continue → 완료 화면 → +3⭐ +50EXP
실패: 피드백 + 힌트 → Try Again → 재시도 (-1💗)

어휘 학습 플로우

📚 어휘 → 카테고리 선택 → Start Learning → 단어 카드 → 🎤 발음 → 다음 단어
10개 완료: 완료 화면 → +10⭐ +100EXP → "Vocabulary Master!" 업적

생명력 소진 플로우

미션 실패 → 💗 = 0 → 경고 표시 → 🏪 보상 상점 → ❤️ 구매 → 미션 재개
