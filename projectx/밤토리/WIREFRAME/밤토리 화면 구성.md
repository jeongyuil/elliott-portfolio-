# 아이 모드 화면 

# SUB1 : 홈 화면
 - 구성요소
	- 포포(또는 루나)가 화면 중앙에 아바타 형태로 나와 있음 
		- 아바타 바로 아래 부분에 나와 아바타와 나눈 대화가 채팅형식의 글로 나옴 
	- 화면 중앙 하단에 마이크 버튼이 있음
		- 마이크 버튼을 누르고 아이의 목소리가 입력되고 포포에게 전달됨 
		- 마이크 버튼을 때면 마이크 비활성화 됨 
	- 화면 상단에는 포포(또는 루나) 와 나의 친밀도 및 아바타의 기본정보가 나와 있음 
	- 화면 하단에는 모험 
- 비고 
	- 이페이지는 캐릭터와 프리토킹이 가능한 공간이며 프로토킹한 내용은 모두 기록 된다.
	- 대화 기록을 바탕으로 아이를 온톨로지화 한다. 

# SUB2 : 모험 
- 구성요소
	- 화면 중간에 아바타들이 있는 공간이 나옴 
		- 포포는 왼쪽, 루나 는 오른쪽에 배치하고 실제 말을 할때는 입모양을 움직임 
		- 장면이 변경 될때면 화면 뒷 배경만 변경됨
		- 바로 아래 부분에 나와 아바타와 나눈 대화가 채팅 형식의 글로 나옴 
	- 화면 중간 아래에 마이크 버튼이 존재
		- 홈 화면과 기능은 동일
	- 화면 상단에는 현재 Stage 의 정보가 표기됨 
		- stage 명, stage 진행률, 루나 포포 와의 친밀도
