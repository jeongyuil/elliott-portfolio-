
# Curriculum Storytelling Prompt Library

## Purpose
This document collects and systematizes prompts used to design and evolve the curriculum story, character behavior, learning flow, and progression logic.
It is intended to be reused, extended, and versioned as part of the FDD.

---

## 1. High-Level Curriculum Design Prompts

### 1.1 Curriculum Arc Definition
**Prompt**
```
Design a 4-week curriculum where each week builds narrative continuity,
progressive skill mastery, and emotional engagement.
Define the learning goal, emotional tone, and learner transformation per week.
```

### 1.2 Learning Outcome Backcasting
**Prompt**
```
Given the final learner capability at the end of Week 4,
backward-design weekly milestones and prerequisite skills.
Ensure each week has a measurable behavioral outcome.
```

## 2. Story & Worldbuilding Prompts

### 2.1 World Setting
**Prompt**
```
Create a fictional-but-grounded learning world where skills are acquired through interaction.
Define rules of the world, progression logic, and constraints that mirror real-world learning.
```

### 2.2 Narrative Consistency Check
**Prompt**
```
Review the curriculum storyline and identify breaks in narrative logic,
motivation, or character behavior across weeks.
Suggest fixes that preserve learning intent.
```

## 3. Character Design & Behavior Prompts

### 3.1 Friend Character — Luna (살아있는 외계 아이)
**Prompt**
```
Design a friend character (Luna) who is a living alien child from outer space.
She is NOT a robot — she is a living being, similar in age to the learner.
She is curious, playful, and makes endearing mistakes about Earth.
The child (Protagonist) teaches Luna, creating a reverse-mentoring dynamic.
Define personality traits, how she expresses genuine emotions (surprise, gratitude),
and how her understanding grows as the child helps her.
```

### 3.2 Coach Character — Popo (비밀 요원 코치)
**Prompt**
```
Design a coach character (Popo) who is a kind-hearted secret agent sent by the government.
Popo loves children and acts as a warm, supportive coach.
Popo addresses the child as "Captain" to reinforce their protagonist identity.
Define emotional responses, coaching style, when to intervene vs. observe,
and how Popo bridges Korean and English for the child.
```

### 3.3 Triad Interaction (Luna + Popo + Child as Protagonist)
**Prompt**
```
Write a dialogue where Luna (friend), Popo (coach), and the child (protagonist/captain)
interact together. Role separation:
- Luna = curiosity & mistakes (English only)
- Popo = coaching & emotional safety (Korean + English)
- Child = protagonist/captain who teaches Luna
The child should feel like the hero of the story, not a passive learner.
```

## 4. Session-Level Story Prompts

### 4.1 Session Opening (Storybook-Style)
**Prompt**
```
Write a session opening using the 3-phase Interactive Story Flow:

1. [Phase 1: Narrator — Storybook Narration]
   - Open with "Once upon a time~" style narration (TTS-only)
   - Recap prior story, set the world, describe the current situation

2. [Phase 2: Transition — Protagonist Entry]
   - Popo/Narrator invites the child INTO the story as the protagonist
   - Use trigger words ("Go!", "Open!", "Start!") to activate participation

3. [Phase 3: Interaction Start]
   - The child (protagonist/captain) meets Luna for the first time this session
   - Luna greets the child with genuine excitement
```

### 4.2 Reflection & Closure
**Prompt**
```
Create a reflective closing scene that reinforces learning,
acknowledges effort, and hints at the next session's challenge.
```

## 5. Skill Integration Prompts

### 5.1 Skill-to-Story Mapping
**Prompt**
```
Given a list of skills, embed each skill naturally into story events,
dialogue, or learner choices without explicit exposition.
```

### 5.2 Difficulty Calibration
**Prompt**
```
Evaluate whether this session's tasks are too easy or too difficult
based on prior skill acquisition. Suggest narrative-based adjustments.
```

---

## 6. QA & Consistency Prompts

### 6.1 Cognitive Load Review
**Prompt**
```
Review this session for cognitive overload.
Identify where instructions, story, or tasks should be simplified or split.
```

### 6.2 Alignment with FDD
**Prompt**
```
Check this curriculum content against the FDD.
Flag missing requirements, misaligned skills, or scope creep.
```
