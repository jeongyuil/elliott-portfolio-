# 🚀 Project: Galaxy Guide Byul (은하 가이드 별이)
> **Product Requirements Document (PRD)**
>
> *   **Version:** 2.0.0 (Triad Model Update)
> *   **Last Updated:** 2025-12-27
> *   **Core Concept:** The Helper Effect + Triad Interaction (Luna-Popo-Byul)

---

## 1. Project Overview (개요)
*   **Concept:** 낯가림 심한 아이(별이)가 엉뚱한 외계인(루나)에게 영어를 가르쳐주는 **"역멘토링(Reverse Mentoring)"** 학습 게임.
*   **Key Update:** 영어 몰입 환경을 위해 **루나는 100% 영어**만 사용하며, 한국어 가이드 로봇 **'포포'가 통역 및 학습 코칭을 담당함.

## 3. Interaction Flow (대화 메커니즘)
* Feedback
	* MVP Scope 은 phase 1 으로
	* 별이의 **개인화 특성**을 알아내기 위해 **튜토리얼**을 활용할 수 있는 방향 추가
	* 루나의 세계관 구체화 **(phase 1)**
		* 루나가 별이에게 질문해서 아이스 브레이킹 할 수 있도록 --> 개인화 정보 수집
		* 어느별에서 무슨 이유 때문에 왔어, 누구랑 왔어 등등 
		* 포포는 지구의 조사관: 개인화 모델 (자비스)
			* 개인화에 따라서 **특성 변화 가능**
			* 별이의 감정 상태 확인 가능
	* 전체적인 컨셉은 유지하고, 개인화 특성을 요소요소에 넣는 방향으로
		* 구체적으로 어느 목적으로 활용할 것인지 고민

시스템은 **[상황 발생] -> [포포의 코칭] -> [별이의 발화] -> [루나의 리액션]** 순서로 작동합니다.

### 🔄 Scenario

### 🟢 Phase 1: Bonding (관계 형성 & 기초 명사) - W1~W4
> **Goal:** 루나와 친구가 되고 입 떼기 (Laquor Building)
*   **W1 [Earth Crew]:** 대원 등록 (Name, Hello)
*   **W2 [Lost Parts]:** 방 안의 우주선 부품 찾기 (In/On/Under, Furniture)
*   **W3 [Rocket Fuel]:** 연료(음식) 구하기 (Colors, Fruits)
*   **W4 [Disguise]:** 지구인 변장 옷 입기 (Clothing, Put on)

### 🔵 Phase 2: Adventure (생활 규칙 & 동사) - W5~W8
> **Goal:** 일상생활 속 문제 해결 (Problem Solving)
*   **W5 [Alien Virus]:** 아픈 펫 치료 (Body parts, "It hurts")
*   **W6 [School Infiltration]:** 학교 규칙 알려주기 (School objects, Sit/Listen)
*   **W7 [Weather Error]:** 날씨 기계 고치기 (Weather, Hot/Cold)
*   **W8 [Galaxy Shop]:** 기념품 가게 쇼핑 (Numbers, Money)

### 🟣 Phase 3: Galaxy Report (표현 & 창작) - W9~W12
> **Goal:** 감정 표현 및 나만의 이야기 만들기 (Storytelling)
*   **W9 [My Crew]:** 가족 소개 (Family, He/She is...)
*   **W10 [Emotion Chip]:** 감정 가르치기 (Emotions, Happy/Sad)
*   **W11 [Dream Planet]:** **[Art]** 내가 그린 우주 설명하기 (Drawing, Imagination)
*   **W12 [Message to Space]:** 지구 가이드북 전송 (Review, Farewell)

### 🎓 Phase 4: Galaxy Academy (사회성 & 리딩) - W13~W16
> **Goal:** 학교 갈 준비 및 파닉스 (School Readiness)
*   **W13 [Hologram Call]:** 외계 친구들과 영상통화 (Likes/Hobbies)
*   **W14 [Show & Tell]:** 교수님이 되어 물건 발표 (Functions, "It is for...")
*   **W15 [Playground Rules]:** 친구 간 갈등 해결 (Apology, Sharing)
*   **W16 [Alien Alphabet]:** **[Phonics]** 외계 암호 해독 (Phonics, Reading)

## 5. Key Features (아이디어어)

### 5.1. The Translator Bar (우주 통역기) --> spec out
*   **UI:** 화면 하단 에너지 바 형태.
*   **Logic:** 아이가 말을 하면 게이지가 차오르고, 꽉 차면 루나가 "아하! 그런 뜻이구나!"라며 이해함. (시각적 발화 보상)

### 5.2. Living Sketchbook (살아있는 스케치북) --> spec out
*   **Tech:** Vision API + AR
*   **Logic:** 아이가 그린 그림을 카메라로 비추면 3D 객체로 변환되어 화면 속 루나와 상호작용함.

### 5.3. Alien Code Decoder (외계 암호 해독기) --> spec out
*   **Tech:** STT + Phonics Engine
*   **Logic:** 화면에 뜬 외계어(알파벳)를 소리 내어 읽으면 지구의 그림(단어)으로 변신.

## 6. 시스템 연동 및 분석 (Backend)

### 6-1. 데이터 분석 엔진 (Analytics)
* **발화량 추적:** 세션당 별이의 영어/한국어 발화 비율 모니터링
* **어휘 성장도:** 미션 수행 중 새롭게 사용한 단어 수 기록
* **자신감 지수:** 루나의 실수를 정정해 줄 때의 음성 톤과 속도 분석

### 6-2. 개인화 피드백 (Tier 2 Standard Plan)
* **주간 리포트:** 별이가 루나를 어떻게 도와주었는지 구체적인 에피소드 중심 리포트 발송
* **AI 코디네이터:** 루나 캐릭터가 직접 등장하여 "별이야 고마워!"라고 말하는 감사 영상 제공

### 6-3. 멀티모달 인터페이스
* 음성 대화와 동시에 태블릿 화면에 유니콘, 학교, 시장 등의 **시각적 힌트**를 노출하여 학습 몰입도 강화


**본 커리큘럼은 권별 어린이의 관심사와 발달 단계에 맞춰 실시간으로 난이도가 조정됩니다.**

## 7. 학부모 대시보드 (Parent Insight System)

부모의 불안을 해소하고 효능감을 주는 데이터 시각화 서비스입니다.

### **A. Daily: 오늘의 대화 하이라이트**
* **기능:** 20분 대화 중 아이가 가장 논리적이거나 창의적으로 말한 순간을 **30초 오디오 클립**으로 제공.
* **가치:** "우리 아이가 이렇게 말을 잘해?"라는 안도감과 발견의 기쁨 제공.

### **B. Weekly: 어휘 우주 지도 (Word Cloud)**
* **시각화:** 아이가 사용한 단어를 행성계처럼 시각화 (중심 단어 + 확장 단어).
* **지표:** $$ \text{Vocabulary Richness Score} = \frac{\text{Unique Words}}{\text{Total Words}} $$ (또래 상위 % 비교).

### **C. Monthly: 초등 입학 준비도 지수**
* **분석 항목:**
    1. **지시 이행력:** 질문 의도 파악 능력.
    2. **문장 완결성:** 주어-서술어 호응도.
    3. **발음 정확도:** 제3자 전달력 (STT 신뢰도 기반).
* **Action Item:** 부족한 영역(예: 발음) 발견 시 집에서 할 수 있는 '놀이 키트' 추천.

## 8. 기술적 고려사항 (Tech & Safety)

* **Child-Specific STT:** 성인 데이터가 아닌 아동 음성 데이터로 파인튜닝된 모델 사용 (고음, 부정확한 발음 보정).
* **Safety Guardrail:**
    * **PII Masking:** 주소, 전화번호 발화 시 즉시 마스킹 처리 및 저장 금지.
    * **Red Flag Alert:** "무서워", "싫어", "때렸어" 등 위기 키워드 반복 감지 시 즉시 부모에게 푸시 알림 발송.
