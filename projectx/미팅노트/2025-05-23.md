# 브레인 스토밍  
### 개인 성향 정보를 분석하는 방법
	- SNS 분석을 통해 개인정보를 벡터화 저장 
		- 분석 가능 정보 
			- 이미지를 분석을 통해 성향 분석 
			- tag 분석을 통해 성향 분석 
			- 팔로잉 하고 있는 인친을 분석해서 성향 분석 
	- MBTI 지문 검사 
		- 이미 본인이 알고 있는 MBTI 입력 
		- 추가적인 MBTI 관련 질문 입력 을 통해 모든 항목에 대한 퍼센타일 분석 
	- 신체나이 측정? 
		- 건강검진 기록 표를 올릴수 있다 
		- 체지검사 정보방 검사 
		- 검진표 측정 
	- 성병검사 결과 
### 등급제도는 필수 
	- 매칭 상대화 대화 이후에 대화 내용을 분석해서 매칭률을 리포트 받을수 있는 시스템 
### 음성 사서함 기반 소통 
- 삐삐감성을 이용해 실시간 채팅이 아닌 음성 메모를 전달 함
- 대신 일본어 <-> 한국어 로 변환처리 해서 메세지 전달 
	- 음성의 보이스를 그대로 유지한체 언어 변경 처리 방식 
		- 짧은 조사결과 일 <-> 한 음성 변역 모델은 없는것으로 보임 
		- 대신 tts 를 이용한 방식 연구 
		- 이를 위해서는 적절한 llm model 연구 개발이 필수
