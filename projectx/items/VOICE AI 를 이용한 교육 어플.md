# 서비스 컨셉
	- 개요
		- VOICE AI 어플 혹은 디바이스 를 이용해 
		- 4~12세 대상으로 하는 구독형 놀이/교육/진단 시스템 
		- 
	- 형태 : 어플 혹은 디바이스 를 이용한 구독형 서비스
	- 타켓층 : 
		- 연령 : 유아~ 초등학교 저학년 을 둔 학부모 
		- 상세 타켓 
			- 언어 발달이 미숙한 아이의 언어 치료 목적 
			- 제2 외국어 노출을 높히고 싶은 요청 
	- 서비스 형태
		- 전화 영화와 같이 정해진 시간에 특정 주제로 AI 튜터(혹은 반려 AI)와 이야기 하게함 
		  (20분정도 예상)
		- 대화의 모든 내용을 텍스트로 변환 -> 변환된 데이터를 팩트 테이블 형태로 저장 
			- 보관기간 : 영구보관 
			- 데이터 활용처
				- 1. 대화 내용을 분석해서 현재 학습 진행 상태 및 학습 현황 분석 데이터로 이용 
				- 2. 이용자의 개인 정보 / 성향 / 선호 학습 방법 을 AI 가 파악 가능 
				- 3.  
		- 특정주기(한달)로 아이의 발달 상황에 대해 피드백 제공 
			- 옵션1 -  PDF 보고서 형태로 전달 
			- 옵션2 - AI 코디네이터가 분석해주는 영상을 볼수 있음 
			- 옵션3 - 사람 코디네이터가 분석해주는 형태 
	- 서비스 강점 
		- 교육 + 놀이 -> 진단 -> 피드백 제공 의 선순환 구조를 제공 
			- 실제 가시성 있는 리포트를 제공 하여 우리 아이의 상태를 한눈에 파악하기 좋음
			- 데이터가 쌓일수록 아이들의 비교지표에 대한 리포트를 제공해 줄수 있음
