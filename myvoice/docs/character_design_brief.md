# 밤토리 캐릭터 디자인 브리프

## 프로젝트 개요

- **앱**: 밤토리 (MyVoice) — 3~7세 아동 영어 음성 학습 앱
- **테마**: 밤하늘 우주 모험
- **톤**: 따뜻하고 안전한, 그러면서도 모험적인
- **애니메이션 도구**: Rive (상태 기반 전환)
- **사용 환경**: React Native 모바일 앱

---

## 캐릭터 1: 포포 (Popo)

### 성격 키워드
`따뜻한` `든든한` `유쾌한` `인내심` `리더십` `장난기`

### 역할
- 아이의 **미션 파트너이자 가이드**
- 항상 먼저 시범을 보이고, 아이가 따라할 수 있게 이끔
- 루나를 보호하면서 아이에게 "캡틴" 역할을 부여
- 위기 상황에서도 유머를 잃지 않는 안정감

### 성격 상세
- "포포가 먼저 해볼게~" — 매번 시범을 보이는 헌신적 가이드
- 과장된 리액션으로 아이를 웃기고 긴장을 풀어줌 ("세상에!", "하하!")
- 루나가 울거나 무서워하면 "울지 마! 포포가 있잖아" 하며 보호
- 아이를 "캡틴"이라 부르며 자존감을 높여줌

### 외형 방향

| 항목 | 설명 |
|------|------|
| **종류** | 작고 동글동글한 우주 생물 (포유류 느낌) |
| **체형** | 둥글고 폭신한 체형, 안아주고 싶은 실루엣 |
| **크기 비율** | 머리:몸 = 1:1 또는 2:1 (SD 비율) |
| **색상** | 따뜻한 계열 — 밤색/갈색/베이지 + 별빛 포인트 (금색/노란색) |
| **눈** | 크고 반짝이는 눈, 다양한 감정을 표현할 수 있는 눈썹 |
| **특징** | 작은 안테나 또는 별 모양 장식 (우주 요원 느낌) |
| **악세사리** | 작은 배지 또는 스카프 (비밀 요원 코드) |
| **질감** | 부드럽고 포근한 느낌 (봉제인형 같은) |

### 레퍼런스 스타일
- 카카오프렌즈 라이언의 든든함
- 포켓몬 이브이의 동글동글함
- 두들리(Duolingo 올빼미)의 친근함
- **절대 아닌 것**: 날카로운 선, 어두운 색, 무서운 표정

### 필요 표정/포즈 (8종)

| 상태 | 설명 | 사용 화면 |
|------|------|----------|
| **idle** | 미소 짓고 대기, 살짝 흔들림 | 아이 홈 기본 |
| **talking** | 입 움직임 + 손짓 | 시범 보일 때 |
| **listening** | 귀 기울이기, 눈 반짝 | 아이가 말할 때 |
| **happy** | 뛰어오르며 박수 | 정답/칭찬 |
| **surprised** | 눈 크게 뜨기, "세상에!" | 새 발견 시 |
| **encouraging** | 엄지척 + 미소 | 힌트/격려 |
| **thinking** | 머리 기울이기 | 질문할 때 |
| **celebrating** | 별빛 이펙트 + 점프 | 미션 완료 |

## 캐릭터 2: 루나 (Luna)

### 성격 키워드
`호기심` `순수함` `용감함` `서툰 매력` `감성적` `성장`

### 역할
- 아이와 **함께 영어를 배우는 또래 친구**
- 우주에서 온 외계인이라 지구의 모든 것이 신기함
- 영어가 서툴어서 아이가 가르쳐주는 구조 → 아이의 자신감↑
- 실수하고 넘어지지만 다시 일어나는 모습으로 "틀려도 괜찮아" 메시지

### 성격 상세
- "H-hello... I am Luna. I am... a little scared." — 처음엔 겁먹은 작은 외계인
- 지구 물건에 감탄 ("So soft! In my spaceship, I sleep on hard metal!")
- 실수가 웃기고 사랑스러움 (바지를 머리에 쓰고, 양말을 손에 끼고)
- 감정에 솔직 — 슬프면 파란 물방울, 기쁘면 눈이 반짝
- 시리즈가 진행될수록 자신감이 생기고 문장이 길어짐

### 외형 방향

| 항목 | 설명 |
|------|------|
| **종류** | 작은 우주인/외계 생물 (인간형에 가까운) |
| **체형** | 포포보다 약간 작고 가녀린 느낌 |
| **크기 비율** | 머리:몸 = 2:1 (큰 머리, 큰 눈 강조) |
| **색상** | 시원한 계열 — 라벤더/연보라/은빛 + 달빛 포인트 (흰색/은색) |
| **눈** | 매우 크고 반짝이는 눈, 감정에 따라 색이 미세하게 변함 |
| **특징** | 작은 더듬이 또는 빛나는 귀, 감정 표시 장치 (가슴의 작은 화면) |
| **악세사리** | 작은 우주복 요소 (헬멧? 부츠?) — 약간 헐렁한 느낌 |
| **질감** | 매끈하고 반투명한 느낌 (달빛/별빛 반사) |

### 레퍼런스 스타일
- 스티치(Stitch)의 서툰 매력 (약간 순화 버전)
- 피카츄의 표현력
- 안에서 빛이 나는 느낌 (반딧불 같은)
- **절대 아닌 것**: 어른스러운 표정, 날카로운 이빨, 그로테스크

### 필요 표정/포즈 (8종)

| 상태 | 설명 | 사용 화면 |
|------|------|----------|
| **idle** | 두리번두리번, 호기심 가득 | 모험 화면 기본 |
| **talking** | 서툰 입 움직임, 손 흔들며 | 영어 말할 때 |
| **listening** | 고개 기울이기, "?" 이모티콘 | 포포/아이 말 들을 때 |
| **happy** | 온몸이 빛남, 뛰어오름 | 새로운 단어 배울 때 |
| **scared** | 움츠리기, 눈 크게 | 무서운 상황 |
| **sad** | 파란 물방울, 고개 숙임 | 우주선 파츠 못 찾을 때 |
| **confused** | 머리 위 물음표, 갸우뚱 | 이해 못 할 때 |
| **grateful** | 두 손 모으기, 눈 반짝 | 아이에게 감사할 때 |

## 캐릭터 관계 & 화면 배치

### 관계 다이어그램

```
   [캡틴 (아이)]
      ↑ 가르쳐주는 관계
      |
   [루나] ←——보호——→ [포포]
   (배우는 친구)    (가이드)
```

### 크기 비율 (화면 내)

```
포포: 100% (기준)
루나: 85~90% (포포보다 약간 작음)
```

### 화면별 배치

#### 아이 홈 (#7)
```
┌─────────────────┐
│   [포포 상반신]   │  ← 화면 40% 차지
│                  │
│  ┌─────────────┐ │
│  │  대화 버블   │ │  ← 말풍선 (포포 위 또는 옆)
│  └─────────────┘ │
│                  │
│  [대화 로그 영역] │  ← 스크롤 가능
│                  │
│     🎤 마이크    │  ← 하단 고정
└─────────────────┘
```

#### 모험 플레이 (#8)
```
┌─────────────────┐
│  STAGE 1-3  ━━○ │  ← 스테이지 헤더
│                  │
│ [포포]    [루나] │  ← 듀얼 캐릭터 (양쪽)
│                  │
│  ┌─────────────┐ │
│  │ 대화/자막    │ │  ← 중앙 대화 영역
│  └─────────────┘ │
│                  │
│  🎤    ⏭️  ❌   │  ← 하단 컨트롤
└─────────────────┘
```

#### 대화 중 상태 전환
```
포포 시범:   포포=talking, 루나=listening
아이 차례:   포포=listening, 루나=listening (기대하는 눈)
아이 성공:   포포=happy, 루나=happy
아이 재시도: 포포=encouraging, 루나=confused→happy
```

## 디자인 원칙

1. **3세도 구별 가능** — 실루엣만으로 포포/루나를 구별할 수 있어야 함
2. **감정이 1초 안에 읽힘** — 표정 변화가 명확하고 과장되어야 함
3. **밤하늘에서 돋보임** — 어두운 배경에서 캐릭터가 빛나는 느낌
4. **터치 친화적** — 60px 이상의 터치 영역, 둥근 히트박스
5. **일관성** — 모든 포즈에서 같은 캐릭터로 인식되어야 함
