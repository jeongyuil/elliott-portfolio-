# 밤토리 와이어프레임 문서

**작성일:** 2026년 2월 13일  
**버전:** 1.0

---

## 개요

이 문서는 밤토리 영어 학습 앱의 주요 화면 와이어프레임을 설명합니다. 각 화면의 레이아웃, 구성 요소, 사용자 인터랙션을 시각적으로 표현하여 개발자와 디자이너가 일관된 UI를 구현할 수 있도록 돕습니다.

---

## 와이어프레임 목록

### 1. 홈 화면 (Home Screen)

**파일:** `wireframe-home.png`

**화면 구성:**

홈 화면은 사용자가 앱을 열었을 때 가장 먼저 보는 화면으로, 학습 현황과 주요 기능에 빠르게 접근할 수 있도록 설계되었습니다.

**상단 영역 (Header)**
- 좌측: 사용자 프로필 아이콘 (원형)
- 우측: 하트 아이콘 3개 (현재 보유 하트 표시)

**스트릭 카드 (Streak Card)**
- 불꽃 아이콘과 함께 연속 로그인 일수 표시
- 예: "5 DAYS STREAK"
- 배경: 그라디언트 카드 스타일

**오늘의 미션 (Today's Missions)**
- 섹션 제목: "TODAY'S MISSIONS"
- 3개의 미션 카드를 세로로 나열
- 각 미션 카드 구성:
  - 좌측: 미션 아이콘 (정사각형)
  - 중앙: 미션 제목 (1줄) + 설명 (1줄)
  - 하단: 진행 상황 프로그레스 바

**주간 목표 (Weekly Goals)**
- 섹션 제목: "WEEKLY GOALS"
- 4개의 목표 카드를 2x2 그리드로 배치
- 각 목표 카드 구성:
  - 상단: 아이콘 (XP, 미션, 시간, 단어)
  - 중앙: 목표 이름 (1줄)
  - 하단: 진행 상황 프로그레스 바

**하단 네비게이션 (Bottom Navigation)**
- 5개의 탭: Home, Missions, Shop, Skills, Profile
- 각 탭에 아이콘과 레이블 표시
- 현재 활성 탭은 강조 표시

**디자인 특징:**
- 카드 기반 레이아웃으로 정보를 명확히 구분
- 프로그레스 바로 진행 상황을 직관적으로 표시
- 고정된 하단 네비게이션으로 빠른 화면 전환

### 2. 미션 화면 (Missions Screen)

**파일:** `wireframe-missions.png`

**화면 구성:**

미션 화면은 사용자가 완료할 수 있는 다양한 학습 미션을 목록 형태로 제공합니다.

**상단 영역 (Header)**
- 좌측: 뒤로 가기 버튼 (화살표 아이콘)
- 중앙: "Missions" 제목
- 우측: 필터 아이콘 (미션 정렬/필터링)

**미션 목록 (Mission List)**
- 세로 스크롤 가능한 미션 카드 리스트
- 각 미션 카드 구성:
  - 좌측: 미션 아이콘 (원형, 미션 타입 표시)
  - 중앙 상단: 미션 제목 (예: "Complete Onboarding")
  - 중앙 하단: 미션 설명 (예: "Set up your profile and preferences.")
  - 우측 상단: 난이도 배지 (EASY, MEDIUM, HARD)
  - 우측 하단: XP 보상 (예: "+50 XP")
  - 하단: 진행 상황 표시
    - 완료된 미션: "Completed" + 체크마크
    - 진행 중 미션: "Progress" + 프로그레스 바
    - 새 미션: "New" + 빈 프로그레스 바

**미션 상태:**
- 완료: 어두운 프로그레스 바 + 체크마크
- 진행 중: 부분적으로 채워진 프로그레스 바
- 새 미션: 빈 프로그레스 바

**하단 네비게이션 (Bottom Navigation)**
- 5개의 탭 표시 (Missions 탭 활성화)

**디자인 특징:**
- 미션 상태를 색상과 아이콘으로 명확히 구분
- 난이도와 보상을 한눈에 파악 가능
- 스크롤 가능한 목록으로 많은 미션 수용

### 3. 프로필 화면 (Profile Screen)

**파일:** `wireframe-profile.png`

**화면 구성:**

프로필 화면은 사용자의 학습 통계와 업적을 확인할 수 있는 화면입니다.

**상단 영역 (Header)**
- 중앙: 대형 프로필 아바타 (원형)
- 아바타 하단: 사용자 이름 (1줄)
- 이름 하단: 레벨 배지 (예: "Level 45")
- 우측 상단: 로그아웃 버튼

**통계 카드 (Stats Cards)**
- 3개의 통계를 가로로 나열
- 각 통계 구성:
  - 상단: 숫자 (예: "1.2k", "95", "7")
  - 하단: 아이콘 + 레이블 (Stars, Hearts, Streak)

**주간 진행 상황 (Weekly Progress)**
- 섹션 제목: "Weekly Progress"
- 4개의 진행 카드를 2x2 그리드로 배치
- 각 진행 카드 구성:
  - 좌측: 아이콘 (별, 트로피, 시계, 말풍선)
  - 우측 상단: 카테고리 이름 (XP, Missions, Study Time, Words)
  - 우측 중앙: 현재/목표 값 (예: "4500/5000", "12/20", "8h 30m", "250")
  - 하단: 프로그레스 바

**업적 (Achievements)**
- 섹션 제목: "Achievements"
- 3개의 배지를 가로로 나열
- 각 배지는 원형 메달 형태로 아이콘 표시

**하단 네비게이션 (Bottom Navigation)**
- 5개의 탭 표시 (Profile 탭 활성화)

**디자인 특징:**
- 대형 아바타로 개인화된 느낌 제공
- 통계를 카드로 구분하여 가독성 향상
- 배지로 성취감 시각화

### 4. 어휘 학습 화면 (Vocabulary Screen)

**파일:** `wireframe-vocabulary.png`

**화면 구성:**

어휘 학습 화면은 카테고리별로 단어를 학습할 수 있는 화면입니다.

**상단 영역 (Header)**
- 좌측: 뒤로 가기 버튼
- 중앙: "Vocabulary" 제목
- 우측: 검색 아이콘

**카테고리 탭 (Category Tabs)**
- 가로 스크롤 가능한 탭 바
- 탭 목록: Animals, Food, Colors, Numbers
- 활성 탭은 어두운 배경으로 강조

**단어 카드 그리드 (Word Card Grid)**
- 2열 그리드 레이아웃
- 세로 스크롤 가능
- 각 단어 카드 구성:
  - 상단: 이미지/아이콘 플레이스홀더 (정사각형)
  - 중앙 상단: 단어 (1줄)
  - 중앙 하단: 발음 기호 (1줄)
  - 우측 하단: 오디오 재생 버튼 (스피커 아이콘)

**하단 버튼 (Bottom Button)**
- 전체 너비의 "Practice" 버튼
- 네비게이션 바 위에 배치

**하단 네비게이션 (Bottom Navigation)**
- 5개의 탭 표시

**디자인 특징:**
- 카테고리 탭으로 단어를 효율적으로 분류
- 2열 그리드로 많은 단어를 한 화면에 표시
- 오디오 버튼으로 발음 학습 지원
- Practice 버튼으로 학습 모드 전환

### 5. 온보딩 튜토리얼 (Onboarding Tutorial)

**파일:** `wireframe-onboarding.png`

**화면 구성:**

온보딩 튜토리얼은 신규 사용자를 위한 앱 사용법 안내 모달입니다.

**배경 (Background)**
- 어두운 오버레이로 기존 화면을 반투명하게 가림

**모달 카드 (Modal Card)**
- 중앙에 배치된 둥근 모서리 카드
- 우측 상단: X 닫기 버튼

**모달 내용 (Modal Content)**
- 상단: 대형 원형 아이콘 플레이스홀더
  - 그라디언트 배경으로 시각적 강조
- 중앙 상단: 제목 (예: "Onboarding tutorias")
- 중앙 하단: 설명 텍스트 (2-3줄)
  - Lorem ipsum 플레이스홀더 텍스트

**진행 표시 (Progress Indicators)**
- 5개의 점(dot)을 가로로 나열
- 현재 단계는 채워진 점으로 표시
- 예: ○ ● ○ ○ ○ (2번째 단계)

**네비게이션 버튼 (Navigation Buttons)**
- 하단에 2개의 버튼을 나란히 배치
- 좌측: "Previous" 버튼
- 우측: "Next" 버튼
- 버튼은 투명 배경에 테두리 스타일

**디자인 특징:**
- 모달 오버레이로 집중도 향상
- 진행 표시 점으로 전체 단계 파악
- 이전/다음 버튼으로 자유로운 탐색
- X 버튼으로 언제든지 건너뛰기 가능

## 공통 디자인 원칙

### 레이아웃

모든 화면은 다음 레이아웃 원칙을 따릅니다.

**고정 요소:**
- 상단 헤더: 뒤로 가기, 제목, 액션 버튼
- 하단 네비게이션: 5개의 주요 탭

**스크롤 영역:**
- 헤더와 네비게이션 사이의 모든 콘텐츠는 세로 스크롤 가능
- 스크롤바는 화면에 노출되지 않음 (숨김 처리)

**카드 기반:**
- 정보를 카드 단위로 그룹화
- 카드 간 충분한 여백 확보
- 둥근 모서리로 부드러운 느낌

### 타이포그래피

**제목 (Headings):**
- 대문자 사용 (예: "TODAY'S MISSIONS")
- 굵은 폰트 (Bold)
- 섹션 구분을 명확히

**본문 (Body Text):**
- 일반 폰트 (Regular)
- 충분한 줄 간격
- 가독성 우선

**숫자 (Numbers):**
- 큰 크기로 강조
- 통계 및 진행 상황 표시

### 아이콘

**스타일:**
- 선 기반 아이콘 (Line Icons)
- 일관된 선 두께
- 원형 또는 정사각형 컨테이너

**용도:**
- 네비게이션: 집, 체크마크, 장바구니, 졸업모자, 사람
- 통계: 별, 하트, 불꽃, 시계, 트로피
- 액션: 뒤로 가기, 검색, 필터, 오디오, 닫기

### 색상

와이어프레임은 그레이스케일로 표현되지만, 실제 구현 시 다음 색상 체계를 사용합니다.

**주요 색상:**
- 듀오링고 그린: 액션 버튼, 성공 상태
- 듀오링고 블루: 링크, 정보 강조
- 듀오링고 옐로우: 별, 보상
- 듀오링고 레드: 하트, 경고

**중립 색상:**
- 배경: 밝은 회색 (#F7F7F7)
- 카드: 흰색 (#FFFFFF)
- 텍스트: 어두운 회색 (#3C3C3C)
- 테두리: 연한 회색 (#E5E5E5)

### 간격 (Spacing)

**패딩:**
- 화면 좌우: 16px
- 카드 내부: 16px
- 섹션 간: 24px

**마진:**
- 카드 간: 12px
- 요소 간: 8px

### 인터랙션

**터치 영역:**
- 최소 44x44px (iOS 가이드라인)
- 버튼과 카드는 충분한 터치 영역 확보

**피드백:**
- 버튼 탭 시 시각적 피드백 (색상 변화)
- 프로그레스 바는 애니메이션으로 채워짐
- 모달은 페이드 인/아웃 애니메이션

**제스처:**
- 세로 스크롤: 모든 목록 화면
- 가로 스크롤: 카테고리 탭
- 탭: 버튼, 카드, 네비게이션
- 스와이프: 모달 닫기 (선택적)

## 반응형 디자인

### 모바일 우선

와이어프레임은 모바일 화면(375x667px 기준)을 우선으로 설계되었습니다.

### 태블릿 적응

태블릿 화면(768px 이상)에서는 다음과 같이 적응합니다.

**레이아웃 변경:**
- 2열 그리드를 3-4열로 확장
- 카드 크기 증가
- 좌우 여백 증가

**네비게이션:**
- 하단 네비게이션을 좌측 사이드바로 변경 (선택적)

### 데스크톱 적응

데스크톱 화면(1024px 이상)에서는 다음과 같이 적응합니다.

**레이아웃 변경:**
- 최대 너비 제한 (1200px)
- 중앙 정렬
- 좌측 사이드바 네비게이션

## 접근성 (Accessibility)

### 색상 대비

모든 텍스트는 WCAG AA 기준 이상의 색상 대비를 유지합니다.

- 일반 텍스트: 4.5:1 이상
- 큰 텍스트: 3:1 이상

### 키보드 탐색

모든 인터랙티브 요소는 키보드로 접근 가능합니다.

- Tab 키로 요소 간 이동
- Enter/Space 키로 활성화
- Esc 키로 모달 닫기

### 스크린 리더

모든 UI 요소는 적절한 레이블과 설명을 제공합니다.

- 이미지: alt 텍스트
- 버튼: aria-label
- 진행 상황: aria-valuenow, aria-valuemin, aria-valuemax

## 구현 가이드

### 우선순위

와이어프레임을 구현할 때 다음 순서를 권장합니다.

1. **1단계: 홈 화면** - 가장 중요한 화면, 모든 주요 기능 접근
2. **2단계: 프로필 화면** - 사용자 정보 및 통계 표시
3. **3단계: 미션 화면** - 핵심 학습 기능
4. **4단계: 온보딩 튜토리얼** - 신규 사용자 경험
5. **5단계: 어휘 학습 화면** - 추가 학습 기능

### 컴포넌트 재사용

다음 컴포넌트는 여러 화면에서 재사용됩니다.

**카드 (Card):**
- 미션 카드
- 목표 카드
- 통계 카드
- 단어 카드

**프로그레스 바 (Progress Bar):**
- 미션 진행 상황
- 주간 목표 진행 상황

**버튼 (Button):**
- 주요 액션 버튼
- 네비게이션 버튼
- 아이콘 버튼

**네비게이션 (Navigation):**
- 하단 탭 바
- 헤더 (뒤로 가기, 제목, 액션)

### 테스트

구현 후 다음 사항을 테스트합니다.

**시각적 테스트:**
- 모든 요소가 와이어프레임과 일치하는지 확인
- 간격과 정렬이 일관적인지 확인
- 다양한 화면 크기에서 테스트

**인터랙션 테스트:**
- 모든 버튼이 작동하는지 확인
- 스크롤이 부드러운지 확인
- 애니메이션이 자연스러운지 확인

**접근성 테스트:**
- 스크린 리더로 테스트
