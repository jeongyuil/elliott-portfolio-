# 밤토리 앱 - 목표 추천 기능 테스트 노트

## 테스트 일시
2026-02-08

## 테스트한 기능
주간 학습 목표 AI 추천 시스템

## 테스트 결과
✅ **성공**: 모든 기능이 정상 작동

### 1. AI 추천 목표 배너
- 목표 설정 모달 상단에 파란색-보라색 그라데이션 배너 표시
- "과거 학습 기록을 분석하여 최적의 목표를 추천합니다" 설명 문구
- "자세히 보기" 버튼으로 추천 상세 내용 토글

### 2. 추천 알고리즘
- 과거 4주 학습 데이터 분석
- 최근 2주 평균 계산
- 증가 추세 감지 시 15% 성장 목표, 안정/감소 추세 시 10% 성장 목표
- 각 지표별 상한선 설정 (XP: 2500, 미션: 35, 시간: 25, 단어: 300)

### 3. 추천 값 (테스트 데이터 기준)
- 획득 XP: 1323 (최근 평균 1150 기준 15% 증가)
- 완료 미션: 23개 (최근 평균 20 기준 15% 증가)
- 학습 시간: 13.5시간 (최근 평균 11.65 기준 15% 증가)
- 학습 단어: 169개 (최근 평균 147 기준 15% 증가)

### 4. 추천 이유 표시
각 목표마다 구체적인 추천 이유를 텍스트로 제공:
- "최근 XP 획득이 증가 추세입니다! 지난 주 평균(1150XP)보다 15% 높은 목표를 추천합니다."
- "미션 완료가 꾸준히 늘고 있어요! 지난 주 평균(20개)보다 15% 높은 목표를 추천합니다."

### 5. 추천 목표 적용
- "✨ 추천 목표 적용하기" 버튼 클릭 시 모든 입력 필드에 추천 값 자동 입력
- 입력 필드 값이 1500 → 1323, 25 → 23, 15 → 13.5, 200 → 169로 변경됨

### 6. 목표 검증 (구현됨, 미테스트)
- 사용자가 입력한 값이 너무 높거나 낮을 경우 경고 메시지 표시
- 최근 평균 대비 0.5배 미만 또는 2.5배 초과 시 경고
